# 🛒 E-Commerce Sales Analysis Dashboard

An end-to-end Data Analytics project that analyzes e-commerce sales data using Python, SQL, Power BI, Streamlit, and Machine Learning. The project provides business insights, interactive dashboards, and future sales forecasting.

---

# 📌 Project Overview

This project focuses on analyzing historical e-commerce transactions to discover valuable business insights. It includes data cleaning, exploratory data analysis (EDA), SQL analysis, Power BI visualization, an interactive Streamlit dashboard, and sales forecasting using Machine Learning.

---

# 🚀 Features

- 📊 Interactive Streamlit Dashboard
- 📈 Monthly Sales Trend Analysis
- 🌍 Country-wise Sales Analysis
- 🏆 Top Selling Products
- 👤 Top Customers Analysis
- 📅 Date Range Filter
- 🔍 Product Search
- 📥 Download Filtered Dataset
- 🤖 Machine Learning Sales Forecast
- 📉 Forecast Visualization
- 📄 Download Forecast Report

# ▶️ Installation

Clone the repository

```bash
git clone https://github.com/93sachin/Ecommerce-Sales-Analysis.git
```

Move into the project

```bash
cd Ecommerce-Sales-Analysis
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Streamlit

```bash
streamlit run app.py
```
